So you have some bitcoins that you want to take "offline", but aren't sure of how to do that or what that even means. Not a problem! This tutorial will guide you through the process of creating a secure, offline wallet to which you can send and safely store your bitcoins.

Step 1: Download Ubuntu

If you are already familiar with Linux and feel like using a different flavor than Ubuntu, feel free to do so. Ubuntu is recommended in this tutorial because it is probably the most accessible for newbies.

You can download the latest release of Ubuntu here. You will be downloading an ISO file, which is an archive of an optical disc. You could use this file to create a CD/DVD, or there are programs that can directly mount an ISO as if it were a physical CD/DVD in your computer's CD/DVD drive.

The download may take a while, so you can move on to the next step while it is downloading.

Step 2: Verify the Ubuntu ISO

After you have finished downloading the Ubuntu ISO, you'll want to verify it by comparing its md5 hash to the one provided to you on the Ubuntu website; you can view the list of hashes here. Find the hash that corresponds to the ISO that you downloaded earlier. These md5 hashes are 32 character length strings of text that were created by passing the contents of each ISO into the md5 hashing algorithm. If you give the md5 algorithm the exact same input data more than once, it will output the exact same 32 character string every time. This is important because this is how you can be sure that the ISO you downloaded has not been tampered with or corrupted in some way.

For Windows Users

If you're on a Windows machine, you can use winmd5 to verify the ISO's md5 hash. It provides a graphical interface for you, so you don't even need to use your command prompt.

Download the winmd5 program, then run it. Select the ISO file you downloaded earlier. Then, enter the md5 hash provided to you from the Ubuntu website in the bottom text field. After the hash has been computed for the ISO file, push the verify button to compare them. If they match, congratulations! You've just successfully verified the ISO file. You can now move on to the next step.

For Linux Users

If you're using some flavor of Linux, you can use the md5sum program to check the md5 hash of the ISO. Navigate to the directory in which you stored the ISO, then run the following command in your terminal:

md5sum ubuntu-14.04-desktop-amd64.iso

The name of the ISO file may be different, so just replace the one above with the file name of the ISO you downloaded earlier.

This may take a little bit of time to process, depending upon the speed of your computer.

Once you've created an md5 hash from the contents of the ISO, compare it to the one you downloaded from the Ubuntu website. If they match, congratulations! You've just successfully verified the ISO file. You can now move on to the next step.

Step 3: Create a Bootable USB Flash Drive

Download UNetbootin for your operating system. This program will allow you to create a bootable USB flash drive from the ISO you downloaded earlier.

Before you can create your bootloader, you'll need to have a USB flash drive (with at least 2GB of space) available. Make sure the drive is formatted as FAT32.

Select "Diskimage" and browse to select the ISO file you downloaded previously. Then select the USB flash drive to which the bootloader will be installed. Finally click the "OK" button to begin the creation of your bootloader. This may take a few minutes.

If you have trouble with this step, you can this tutorial which contains additional information about creating bootable USB flash drives.

Step 4: Get the Bitcoin Wallet Software

Download the bitcoin wallet software for Linux; this will be a .tar.gz file. Once that's done downloading, copy it to the flash drive.

Step 5: Disconnect Your Machine From the Internet

Un-plug your ethernet cable, if there is one.

Now you'll want to disable your wireless adapter. To do this, first restart your computer, but before it boots into your regular, installed operating system, you'll need to open the BIOS menu. Usually this requires you to press the F2 key early on in the boot process. The key to start the BIOS menu may be different for your hardware. Watch carefully when your computer is booting up for instructions for how to get to the BIOS menu.

Once you're in the BIOS menu, navigate through the different sub-menus until you find a reference to your wireless adapter. Disable the wireless adapter.

Do not exit the BIOS just yet! You will need to do one more thing in the BIOS in the next step.

Step 6: Adjust the Boot Order

In order for your machine to boot up using the bootloader installed on the flash drive, you'll need to move the USB media above your primary storage device in the boot order in the BIOS. To do this, navigate to the "Boot Order" sub-menu in the BIOS. Then move the USB media up in the boot order until it is above your primary storage device (which is usually a HDD or SSD).

Once that's done, you can "Save & Exit" from the BIOS.

Step 7: Booting into Ubuntu

With the flash drive still plugged into one of your machine's USB ports, your machine should now automatically boot into the Ubuntu OS that is installed on the flash drive. If it didn't, you'll want to look back over the last few steps to verify you did everything correctly.

Once Ubuntu has started, you'll want to verify that your machine is in fact disconnected from the internet. Open Firefox and try to go to some website. You'll want to see some kind of error page saying that you're not currently online.

If you are able to access the internet, go back to step 5 and make sure you've done everything correctly. If you are not able to access the internet, move on to the next step.

Step 8: Create an Encrypted Bitcoin Wallet

First, you'll need to retrieve the bitcoin wallet software from the flash drive. Open the flash drive up in nautilus (the Ubuntu file manager). Copy the bitcoin .tar.gz file to the desktop. Right click the archive and then "Extract..". Navigate into the folder that this created. Go to bin then 64. Now you will see several different binary files. Double click bitcoin-qt. This is the graphical interface for the bitcoin wallet software.

Encrypt the wallet using a strong password (Settings -> Encrypt wallet). A strong password should consist of alphanumeric characters as well as some non-alphanumeric characters, and it should be at least 20 characters long. This password should be unique. Do not use this password for anything else! Do not save this password to the flash drive or to any electronic device that is not encrypted. You may want to write the password down on a piece of paper and store it in a safety deposit box.

Step 10: Make Note of Receiving Addresses

From within the bitcoin-qt client, you'll want to copy at least a few different public (receiving) bitcoin addresses for your wallet. Paste these addresses into a text document on the flash drive. These addresses will allow you to send bitcoins to this wallet.

Step 9: Backup the Encrypted Wallet File

Open the file manager and navigate to the home directory. Press Ctrl+H to show hidden files and directories. Open the .bitcoin directory. Copy the wallet.dat file to the flash drive.

This is your encrypted wallet file. You should make several backups of this file and preferably store them in multiple different physical locations. If you lose this file (and all copies of it) you will not be able to access the wallet anymore!

Step 11: Clean Up

Shut down your machine and make sure that it is completely powered off. If it's a laptop, take the battery out. Leave the machine sitting like this for at least a minute.

Now you should be able to re-enable your wireless adapter and/or plug the ethernet cable back in, to allow your machine to connect to the internet once again.

Step 12: Sending Bitcoins to the Encrypted Wallet

This step may vary, depending upon where you currently have your bitcoins stored.

Third-Party Wallet Services

If your bitcoins currently live in a web wallet or other third-party wallet service, simply login to your account with that service and send a very small amount of bitcoin to one of the receiving addresses for the encrypted wallet. You only want to send a small amount at this stage because you want to first verify that you didn't make a mistake while creating the encrypted wallet.

Check the receiving address at Block Explorer periodically until you see that the bitcoins have arrived.

Other Wallet Software

If you are currently using some other wallet software (Electrum, Armory, etc.) to store your bitcoins, this will be a bit more complicated. From the machine on which your current wallet is located, copy the wallet.dat file to the flash drive. Be sure not to overwrite the wallet.dat file already there (the one for your new wallet).

Now boot into Ubuntu on the flash drive again. Once Ubuntu has started, you'll need to install the wallet software that contained your old wallet. Once you're done installing that software, navigate to the hidden .bitcoin folder in your the home directory. Replace the wallet.dat found here with the one from your old wallet.

Start the wallet software. Your should now have access to your old wallet. Send a very small amount of bitcoins to one of the receiving addresses for your new wallet.

Check the receiving address at Block Explorer periodically until you see that the bitcoins have arrived.

Step 13: Wrap Up

To be on the safe side, after having sent a small amount of bitcoins to your new wallet, you'll want to try sending a small amount of bitcoins interally to another receiving address of the new wallet. This will ensure that you have control of those bitcoins within the new wallet.

After all that, you can send the rest of your bitcoins to the new wallet. And, if you want to be on the extra safe side, you may want to create several different bitcoin wallets in the same way described in this tutorial. This will allow you to diversify your bitcoin holdings; so that even if one of the wallets is somehow compromised, you won't lose all of your bitcoins.

Keep the flash drive handy. You'll need it whenever you want to securely send bitcoins from your new wallet(s).
